# Understanding AI

Educational SPA for a liberal arts guide to AI — ten sections plus Explore.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Lint

```bash
npm run lint
```

Deployed from repo root via Netlify (`base = understanding-ai`, publish `dist`). Live: https://aiandhumanity.netlify.app/
